# 🎸 guitar-trainer-lib 🎸

Generate random musical keys, notes, and progressions 🎵

## Get started

```javascript
import {
  getRandomProgression,
  getRandomNote,
  getRandomKey
} from 'guitar-trainer-lib'

getRandomProgression() // 'I - V - IV'
getRandomNote() // 'E♭'
getRandomKey() // 'G♭'
```

Happy playing! 🎸🎵
